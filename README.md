#Design of Electro Cart System for Online Shopping (Vision Shop).

	Tech Stack: HTML, CSS, JavaScript, J2EE, Hibernate, MySQL.

	Software used: Eclipse, MySQL Workbench, Chrome.

	Description: Web-based shopping system for the shop. The Concept of the system is to allow the customer to shop virtually. 
The system will provide a security module with four levels of access: Admin, User, Vendor, and Delivery Person. 
Used Java for backend data support and JavaScript with HTML-CSS for frontend data access.
